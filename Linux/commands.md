# 리눅스(Linux) 핵심 명령어 모음

## 1. 파일 및 폴더 조작

### 조회 (ls)
- `ls`: 현재 디렉토리 목록 표시
  - `ls -l`: 상세 정보 표시 (권한, 소유자, 크기 등)
  - `ls -a`: 숨김 파일(.으로 시작) 포함 표시
  - `ls -al`: **(국룰)** 숨김 파일 포함 상세 정보 표시
  - `ls -lh`: 파일 크기를 보기 편한 단위(KB, MB)로 표시
  - `ls -Rt`: 하위 디렉토리까지 포함하여 최신순 정렬

### 이동 (cd)
- `cd [경로]`: 해당 디렉토리로 이동
  - `cd ..`: 상위 디렉토리로 이동
  - `cd ~`: 홈(Home) 디렉토리로 이동
  - `cd -`: 바로 직전에 있었던 디렉토리로 복귀 (뒤로 가기)

### 생성 (mkdir, touch)
- `mkdir [이름]`: 디렉토리 생성
  - `mkdir -p a/b/c`: 하위 디렉토리까지 한 번에 생성 (Parent)
- `touch [이름]`: 빈 파일 생성 또는 날짜 갱신

### 삭제 (rm) - 주의 필요
- `rm [파일]`: 파일 삭제
- `rm -r [폴더]`: 폴더 및 내부 파일 삭제
- `rm -rf [폴더]`: **(주의)** 묻지 않고 강제로 폴더 삭제

### 복사/이동 (cp, mv)
- `cp [원본] [복사본]`: 파일 복사
  - `cp -r [폴더] [복사본]`: 폴더 통째로 복사
- `mv [원본] [목적지]`: 파일 이동
- `mv [이름1] [이름2]`: 파일 이름 변경


## 2. 내용 보기 및 검색

### 파일 읽기 (cat, head, tail)
- `cat [파일]`: 파일 내용 전체 출력
- `head -n 10 [파일]`: 위에서 10줄만 보기
- `tail -n 10 [파일]`: 아래서 10줄만 보기
  - `tail -f [로그파일]`: **(실시간)** 로그가 쌓이는 것을 실시간으로 관전

### 검색 (grep, find)
- `grep "단어" [파일]`: 파일 내 특정 단어 검색
  - `grep -r "단어" .`: 현재 폴더 내 모든 파일에서 찾기
  - `grep -i "단어" [파일]`: 대소문자 구분 없이 찾기
- `find [경로] -name "파일명"`: 파일 위치 찾기
